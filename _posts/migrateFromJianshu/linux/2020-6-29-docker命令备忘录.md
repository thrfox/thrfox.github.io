### docker CLI
进入容器
```
docker ps # 获取容器Id
docker exec -it ${id} /bin/bash  
```
查看容器信息
```
docker logs --tail 50 --follow
# 最后50行
```

### 查看容器所在网络
~~~sh
docker inspect 87236df554a2 -f "{{json .NetworkSettings.Networks }}"
~~~
